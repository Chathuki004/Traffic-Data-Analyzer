# 🚦 Traffic Data Analyzer



## 📊 Overview

A simple Python project that analyzes traffic data from CSV files and helps you understand basic traffic flow patterns.

It reads data, processes it, and gives simple insights in an easy to understand way.



## ✨ Features

- 📂 Reads traffic data from CSV files  
- 🔍 Processes and analyzes the data  
- 📈 Shows basic traffic flow insights  
- 🧑‍💻 Beginner-friendly Python project  



## 📁 Project Setup

- Keep all CSV files in the same folder as the Python file  
- Make sure Python 3 is installed on your system
